Intensity guide (RPE)

Use RPE 7–8 for main sets (2–3 reps in reserve), RPE 6–7 for accessories, RPE 8–9 on finishers.

Progression

  • Weeks 1–2: learn form; land mid rep ranges at RPE ~7–8.
  • Weeks 3–4: add 1–2 reps per set while keeping RPE ≤8.
  • Weeks 5–6: keep reps, increase load (7→9kg, 12→16kg) or add a set to mains.
  • Deload (optional): every 6th week, cut volume by ~40% to freshen up.
  • When kit tops out: slower eccentrics (4–5s), 1–2s pauses, –15–30s rest, or +1 set.
